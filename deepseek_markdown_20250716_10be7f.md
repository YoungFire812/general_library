# Book Exchange Platform (FastAPI)

## Техническое задание

### Стек технологий
- **Backend**: FastAPI
- **База данных**: PostgreSQL (асинхронный доступ через SQLAlchemy 2.0)
- **Фоновые задачи**: Celery + Redis
- **Геоданные**: Google Maps API / OpenStreetMap
- **Фронтенд**: Отдельный SPA (по желанию)
- **Деплой**: Docker + Nginx

---

## Этапы разработки

### 1. Настройка базовой инфраструктуры (3 дня)
- Установка и конфигурация FastAPI
- Подключение к PostgreSQL
- Настройка асинхронного ORM (SQLAlchemy 2.0)
- Базовая система аутентификации (JWT)

### 2. Модуль пользователей (2 дня)
- Регистрация/авторизация
- Профиль пользователя
- Система рейтингов
- Журнал активности

### 3. Система книг (4 дня)
- Добавление/редактирование книг
- Модерация контента (фоновые задачи Celery)
- Статусы видимости (в ленте/только в профиле)
- Категории и теги

### 4. Механика обмена (5 дней)
- Создание предложений обмена
- Система подтверждения/отклонения
- Лимиты (1-5 книг в обмен)
- История операций

### 5. Геосервисы (3 дня)
- Интеграция карт
- Модель постаматов
- Поиск ближайших точек
- Расчет маршрутов

### 6. Логистика (3 дня)
- Система назначения курьеров
- Трекинг доставки
- QR-подтверждение операций
- Арбитраж споров

### 7. Дополнительные функции (2 дня)
- Система жертвования книг
- Алгоритм рекомендаций
- Уведомления (email/websocket)

### 8. Тестирование и деплой (4 дня)
- Нагрузочное тестирование
- Контейнеризация (Docker)
- Развертывание на VPS
- Мониторинг (Sentry)
